This repository contains a comprehensive collection of implementations and research reports for the **Deep Generative Models** course at the University of Tehran. The projects explore the theoretical foundations and practical applications of modern generative architectures, ranging from Variational Autoencoders (VAEs) to Score-Based Generative Models (SGMs).

Each project includes a dedicated report, mathematical derivations, and a self-contained Jupyter Notebook implementation.
